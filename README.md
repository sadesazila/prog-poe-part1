## Project Overview

This is Part 2 of the Cybersecurity Awareness Chatbot POE.
The project expands on Part 1 by introducing a full WPF graphical user interface,
keyword recognition, random responses, memory and recall, sentiment detection,
and improved conversation flow.

---

## Features Implemented

- **GUI Design** - Dark themed WPF application with chat bubbles and colour coding
- **Keyword Recognition** - Recognises cybersecurity topics including password, phishing, privacy, scam, malware, browsing and social engineering
- **Random Responses** - Randomly selects from multiple responses per topic to keep conversations varied
- **Conversation Flow** - Supports follow-up requests such as "tell me more" or "give me another tip"
- **Memory and Recall** - Remembers the user name and favourite topic and personalises responses accordingly
- **Sentiment Detection** - Detects worried, frustrated or curious sentiment and adjusts responses empathetically
- **Error Handling** - Handles unrecognised input gracefully without crashing
- **Code Optimisation** - Uses Dictionary and List collections with OOP principles throughout

---

## How to Run

1. Open the solution in Visual Studio 2022
2. Set CyberBot as the startup project
3. Press F5 or Ctrl+F5 to run
4. Enter your name when prompted
5. Type any cybersecurity topic or type 'help' to see all available topics
